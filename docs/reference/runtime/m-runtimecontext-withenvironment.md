Replaces the environment half of a runtime context.

## Parameters

- `environment`: <code>'nextEnv</code>
  The new application environment.
- `context`: <code><span><a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-runtimecontext-2.html">RuntimeContext</a>&lt;<span>'runtime,&#32;'env</span>&gt;</span></code>
  The source context.

## Returns

A new context with the replaced environment.
